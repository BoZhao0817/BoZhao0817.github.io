## Overview
This is a UX project designed with **Sketch**, containing 19 pages. The main feature of this android application is pomodoro timer. To cultivate user habits, I designed different maps to unlock based on the total time that user focus on the application, if users reach the "1h", "3h", "5h" milestone, the maps of different views will be unlocked automatically. and I named the application "Clance Timer", which means to see the bigger world.
## My Role
- Compared similiar products and determined the target user of the application.
- Designed features of the application and Designed the UI of the application.  
## Analysis (in Chinese)
- 产品定位：效率类APP，基于番茄工作法的习惯养成类APP。
- 关键词：时间管理
- 产品名称：深渊Timer (Glance Timer)
- 设计思路：时间碎片化是一把双刃剑，创意类软件带给我们新奇感和刺激感的同时也偷走了我们的时间。《2019中国数字用户行为变迁专题分析》指出，用户超50%的时间都花在视频、社交类软件上。换言之，在各种信息消磨我们时间的同时，也需要效率类软件来帮助我们摆脱拖延，提高效率，也就是说效率类软件还有很大的市场空间。尤其对于学生和工作党来说，他们更需要效率类软件帮助他们更好的管理学习、工作和娱乐的时间。
深渊Timer是一款类似游戏的番茄钟，除了具备传统番茄钟的功能外，还引入了一个小故事作为背景，吸引用户去探索，从而更好的激励用户。用户通过积累专注时间，帮助故事主人公解锁新地图，探索新的故事线，和主人公一起成长，从而起到培养用户习惯的作用。

- 主要功能：
  - 番茄钟：帮助用户在计时期间保持专注。
番茄钟计时开始，暂停，停止。自动休息，自动进入下一个番茄钟。
  - 里程碑：通过积累时间解锁不同的地图，用户可以看到同一地图下的用户总数，起到很强的激励作用
  - 标签：对番茄钟的使用场景进行分类，如学习，工作，运动。
  - 分析：详尽的报表分析专注时间

- 目标用户：学生党（主要）、工作党。
- 盈利模式：付费下载和内置付费（解锁人物，地图）
- 使用场景：
  - 在需要专注的时候使用番茄钟
  - 查看一段时间内自己的的完成情况
  - 解锁自己的目标，例如专注1000小时（这里我们通过解锁不同的地图来表示）
